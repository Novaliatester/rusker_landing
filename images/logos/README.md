# Logos Directory

This directory should contain the logo images for the "Ils nous font confiance" section.

## Required Logo Files:

1. **essec.png** - ESSEC Business School logo
2. **hec.png** - HEC Paris logo
3. **insead.png** - INSEAD logo
4. **french-tech.png** - French Tech logo
5. **station-f.png** - Station F logo
6. **shoptalk.png** - Shoptalk logo
7. **ai-summit.png** - AI Summit Barcelona logo

## Logo Specifications:

- **Format**: PNG with transparent background
- **Recommended size**: 200-300px width (height will scale proportionally)
- **Color**: Original colors (will be grayscale by default, color on hover)
- **Background**: Transparent

## Where to find logos:

1. **ESSEC**: https://www.essec.edu/en/
2. **HEC Paris**: https://www.hec.edu/
3. **INSEAD**: https://www.insead.edu/
4. **French Tech**: https://www.frenchtech.com/
5. **Station F**: https://stationf.co/
6. **Shoptalk**: https://www.shoptalk.com/
7. **AI Summit Barcelona**: Check official event website

## Note:

The component includes a fallback that will display the organization name as text if the logo image is not found.
